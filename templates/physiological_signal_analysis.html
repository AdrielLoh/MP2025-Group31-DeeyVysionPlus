How the Technology Works

Remote Photoplethysmography

We use 3 different rPPG methods to extract physiological signals via microscopic color fluctuations in the facial skin caused by blood circulation. These subtle changes create unique physiological fingerprints that are difficult to synthesize.

Signal Processing

We then process the extracted rPPG signals to create structured statistics, temporal features, and inter-rPPG method correlations. These statistics and features will then be stacked and passed into an XGBoost model to perform predictions.

AI Pattern Recognition

We have both a deep learning model and a machine learning model, trained on an extensive dataset of authentic and deepfake physiological signals to determine the authenticity of the physioloical features.
