How the Technology Works

Face Detection

Each video frame is scanned to locate faces using a deep neural net-based face detector. Faces are cropped and prepared for model input.

Deep Learning Analysis

Detected faces are passed into a trained convolutional neural network (CNN) which learns to recognize deepfake artifacts and inconsistencies.

Frame-Wise Prediction

Each frame gets a confidence score and label (real or fake). The results are aggregated to make a final prediction for the video.
